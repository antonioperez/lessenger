
# lessenger

Lessenger Chatbot - Weather


### Prerequisites

```
Python 3.6 or later
PIP - python package manager
```



### Installing

```
pip install requirements.txt
```

## Running the tests

```
python tests.py
```


## Running Application

```
python app.py
```
