
# lessenger

Lessenger Chatbot - Weather


### Prerequisites

```
Python 3.6 or later
PIP - python package manager
```

### Installing

```
pip install -r requirements.txt
```

or pip3

## Running the tests

```
python tests.py
```
or python3

## Running Application

```
python app.py
```
